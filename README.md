## Description

This NestJS application offers a powerful backend API with key features like user authentication, user management, and task management. It uses PostgreSQL to store data and provides real-time updates for task creation using WebSocket. The app also includes a Swagger-powered API documentation page, making it easy to explore and interact with the available endpoints.


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
Note: Create an environment filr (.env) to hold the specified env variables for the app
```
DB_HOST=localhost
DB_PORT=database port
DB_USERNAME=database user's username
DB_PASSWORD=database user's password
DB_NAME=database name
JWT_SECRET=your_jwt_secret
```
Swagger API docs at ```localhost:3000/api```
## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
